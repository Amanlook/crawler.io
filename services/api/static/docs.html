Overview

Crawler.io provides a unified REST API to fetch creator profiles, posts, and engagement metrics from Instagram, TikTok, and YouTube through a single normalized interface.

Base URL: http://localhost:8000

Quick Start

# 1. Look up a creator
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "http://localhost:8000/v1/creators/lookup?platform=instagram&username=virat.kohli"

# 2. Get their posts
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "http://localhost:8000/v1/creators/{creator_id}/posts?limit=20"

# 3. Search posts with filters
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "http://localhost:8000/v1/posts/search?platform=instagram&min_likes=1000000"

Authentication

All API requests require a Bearer token in the Authorization header:

Authorization: Bearer sk_test_smoke_test_key_for_local_dev

Enter your API key in the bar above to test endpoints directly from this page. For local development, use:

sk_test_smoke_test_key_for_local_dev

Rate Limits

Rate limits are applied per API key. Limits are returned in response headers.

TierRequests / minDaily Limit
Free10100
Starter10010,000
Pro500100,000
Enterprise2,0001,000,000

Response headers include: X-RateLimit-Limit, X-RateLimit-Remaining, X-RateLimit-Reset

Pagination

List endpoints return a pagination object with cursor-based pagination:

{
  "data": [...],
  "pagination": {
    "has_more": true,
    "next_cursor": "cG9fZ2QzMnk5NDc4...",
    "total_count": 1040
  }
}

To get the next page, pass cursor as a query parameter:

GET /v1/creators/{creator_id}/posts?cursor=cG9fZ2QzMnk5NDc4...&limit=20

Continue until has_more is false.

Creators

Lookup, search, and retrieve creator profiles with follower counts, bios, and engagement metrics.

GET /v1/creators/lookup Look up a creator by platform & username ▶

Returns the full creator profile. If the creator is not yet indexed, returns 202 Accepted and triggers async collection — retry after 10-30 seconds.

Tracking

Subscribe to automatic data collection for creators at configurable frequencies.

POST /v1/tracking/creators Start tracking a creator ▶

Subscribe to automatic data collection for a creator. Supports frequencies: realtime, frequent, standard.

curl -X POST -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"platform": "instagram", "username": "virat.kohli", "frequency": "standard"}' \
  "http://localhost:8000/v1/tracking/creators"
DELETE /v1/tracking/creators/{tracking_id} Stop tracking a creator ▶

Unsubscribe from tracking. The creator data remains in the database.

curl -X DELETE -H "Authorization: Bearer YOUR_API_KEY" \
  "http://localhost:8000/v1/tracking/creators/{tracking_id}"

Webhooks

Register endpoints to receive real-time notifications when tracked creator data changes.

POST /v1/webhooks Register a webhook ▶

Register a URL to receive webhook notifications. Events: creator.updated, post.new, metrics.changed.

curl -X POST -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://your-server.com/webhook", "events": ["creator.updated", "post.new"]}' \
  "http://localhost:8000/v1/webhooks"
DELETE /v1/webhooks/{webhook_id} Delete a webhook ▶

Remove a registered webhook endpoint.

curl -X DELETE -H "Authorization: Bearer YOUR_API_KEY" \
  "http://localhost:8000/v1/webhooks/{webhook_id}"
